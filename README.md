# 立體圖形疊高高

適合小一學生使用的 3D 立體圖形疊放教具。學生可加入正方體、長方體、三角柱和球體，拖動、轉動並測試圖形是否站穩。

## 🎮 遊戲網頁

### [按此開啟「立體圖形疊高高」](https://pakho2433.github.io/P1-Stacking-3D-shapes-high-/)

支援電腦、平板及手機。建議使用 Chrome、Edge 或 Safari 開啟。

## 遊戲功能

- 加入四種立體圖形
- 拖動及疊放圖形
- 轉動或刪除已選圖形
- 測試圖形組合是否站穩
- 穩陣及易跌示範
- 拎起圖形時，陰影會顯示在圖形正下方
- 支援電腦、平板及手機畫面

## 開啟方法

1. 直接按上方「遊戲網頁」連結。
2. 或下載所有檔案後，開啟 `index.html`。
3. 如遊戲網頁尚未啟用，請到 repository 的 **Settings → Pages**，將來源設定為 `main` branch 的根目錄。

## 使用技術

- Three.js
- Cannon.js
- HTML / CSS / JavaScript
